# Contributing

## Reporting bugs
Open an issue with steps to reproduce, your OS, and Python version.

## Feature requests
Open an issue with the feature description and use case.

## Pull requests
1. Fork the repo
2. Create a branch: `git checkout -b feature/your-feature`
3. Commit your changes
4. Push and open a PR against `main`

## Development setup
```bash
npm install
npm run dev
sudo python3 wifi_analyzer_backend/server.py
```
